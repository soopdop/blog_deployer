# One / Another / The other (3개 중)
* One : 불특정한 하나.
* Another : 불특정한 다른 하나. (= an other)
* The other: 나머지 하나. 둘을 빼고 나머지는 정해 졌으니 **The**를 붙인다.

# Some / Others / The others (많은 것 중)
* Some : 일부.
* Others : 나머지 일부.
* The others : 나머지 전체. 정해진 것이니 **The**를 붙인다.
